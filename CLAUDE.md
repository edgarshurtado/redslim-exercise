# Project conventions

# FrontEnd
## FrontEnd Stack
* React
* React Material UI
* Jest for testing
* Vite
* Tailwind

## Specifications
* Avoid creating new components, use MUI whenever possible
* Avoid inline styles and custom classes. Use tailwind. The only exception is the sx prop from MUI

## BackEnd Stack
* Django
* Django Rest Framework
* Pandas
* Pytest
* PostgreSQL
